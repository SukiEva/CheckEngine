ExecDSL
可视化生成器

这是一个单文件静态工具，专门用于快速拼装当前版本的 ExecDSL。左侧编辑结构化表单，右侧实时生成 JSON， 可以直接复制、下载，或者先载入当前参考样例再调整。

Variables

Prechecks

按顺序执行，命中失败后立即停止。

Steps

支持 consumes，会映射为 CTE。

On Fail

生成结果

右侧预览会固定显示，并自动保存到浏览器本地。

          

        


        

          
使用提示

          

            
1. outputs 用英文逗号分隔。

            
2. 变量条件里的 value 会优先按 JSON 值解析，例如数字、布尔、null。

            
3. sql_params 原样保留，适合填 $input.xxx、$context.xxx 这类路径。

            
4. sub_repeat 模板里请保留一段 [] 用于标记重复片段。

            
5. 消息模板支持 f{$path:format_spec}，例如 f{$steps.a.out1:,}。

            
6. 顶层 on_fail 与 precheck.on_fail 一样支持 single / sub_repeat / full_repeat。
